- 公开 `skills.sh` 搜索、安装、更新检查 API。

## 浏览器验证

测试环境：

- 后端：`PORT=7340`
- 前端：`http://127.0.0.1:4179/`
- 数据库：临时 SQLite `/tmp/opendeepsea-skills-live-*/e2e.db`
- skills 管理目录：`/tmp/opendeepsea-skills-live-*/skills`
- local access token：`e2e-token`

浏览器流程：

1. 打开 `/skills`。
2. 在页面加载前写入 `localStorage.opendeepsea.localToken = e2e-token`。
3. 搜索公开 `skills.sh` 关键词 `python`。
4. 安装公开包 `github/awesome-copilot/python-mcp-server-generator`。
5. 触发更新检查并确认结果刷新为“已是最新”。
6. 确认已安装技能列表、安装来源、更新设置、运行历史面板可见。
7. 在 `1440x1000` 与 `390x900` 视口下检查横向溢出。
8. 检查浏览器 `pageerror`、console error、HTTP 4xx/5xx。

结果：

```json

  "desktop": { "scrollWidth": 1440, "clientWidth": 1440 },
```

- `sandbox-exec` 是 macOS 方案；非 macOS 当前 fail closed，不静默降级。
- Vite chunk size warning 仍存在，属于既有打包体积问题，不影响本次验收。
