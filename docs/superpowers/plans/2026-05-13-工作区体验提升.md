# 工作区体验提升 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 提升 OpenClaw Room 的工作区体验，落地三栏布局、`⌘K` 命令面板、可选 `@agent` 选择器、代码块展示、响应式布局和高质量空状态。

**Architecture:** 前端保持现有 React Router 与 React Query 架构，把壳层导航、命令面板、Composer 辅助输入、消息渲染拆成聚焦组件。所有体验增强优先消费现有 API，不新增后端能力；第一轮只做可用体验，不做复杂拖拽、文件引用或主题切换。

**Tech Stack:** React 18、TypeScript、React Router、TanStack Query、TailwindCSS、cmdk、lucide-react、Vite。
