# 公开 skills.sh 在线安装与项目级沙盒脚本执行实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 OpenDeepSea 的 `/skills` 支持公开 `skills.sh` 在线安装、项目级沙盒脚本执行、更新检查与运行审计。

**Architecture:** 保留现有 prompt-only skills 作为兼容层，新增 skills.sh 安装适配器、脚本型 skill manifest、项目级沙盒 runner、运行审计表和前端在线安装/运行记录/更新设置面板。执行型 skill 只在 workflow 或 agent 显式调用时运行，不改变现有内部模型注入逻辑。

**Tech Stack:** TypeScript, Express, SQLite, React 18, Vite, zod, node:test, Lucide icons.
