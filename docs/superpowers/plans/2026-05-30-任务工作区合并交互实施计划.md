# 任务工作区合并交互 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** 按 `docs/superpowers/specs/2026-05-30-任务工作区合并交互设计.md` 将房间页左侧任务栏和右侧任务详情合并为一个任务工作区，并保留 M1-M4 完整任务闭环。

**Architecture:** 新增 `TaskWorkspacePanel` 作为任务 UI 聚合组件，内部拆分任务队列、当前任务头部、执行会话、概览/时间线/变更/日志视图。`RoomPage` 继续持有跨区域状态与查询，聊天主列保留为命令入口，并向 composer 传入明确的任务发送目标提示。

**Tech Stack:** React 18、TypeScript、Vite、Tailwind utility class、现有 CSS token、Node built-in test、React server render test、Playwright/浏览器真实验证。
