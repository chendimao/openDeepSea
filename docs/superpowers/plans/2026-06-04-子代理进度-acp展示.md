# 子代理进度 ACP 展示 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 ACP 时间线和任务事件能够展示结构化子代理进度，并在缺少结构化子代理事件时给出明确诊断。

**Architecture:** 后端在 ACP 事件归一化层新增 `subagent_*` timeline 类型，并将这些事件投影到现有 `task_events.runtime_event`。前端在 `AgentTimeline` 中渲染子代理事件，同时在模型层检测“正文提到子代理但没有结构化事件”的情况并展示诊断。

**Tech Stack:** Node.js、TypeScript、React 18、Vite、SQLite、node:test。

---
