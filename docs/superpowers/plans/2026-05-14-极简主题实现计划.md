# 极简主题 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** 新增一个显示文案为“极简”的独立可切换主题，只改变样式风格，不改变页面布局、DOM 结构或业务行为。

**Architecture:** 使用现有主题机制扩展 `ThemeMode`，在主题切换控件中增加 `minimal` 选项。所有视觉变化集中放在 `packages/frontend/src/index.css` 的 `[data-theme="minimal"]` scoped 规则中，通过 token 和现有类覆盖实现，保证切回其他主题后无残留样式。

**Tech Stack:** React 18、TypeScript、Vite、Tailwind utilities、CSS custom properties、lucide-react。

---

## 文件结构

- Modify: `packages/frontend/src/lib/theme.ts`
  - 负责主题枚举和 localStorage 值校验。
- Modify: `packages/frontend/src/components/AppShell.tsx`
  - 负责主题切换 UI，新增“极简”选项。
- Modify: `packages/frontend/src/index.css`
  - 负责新增 `[data-theme="minimal"]` token 和 scoped 样式覆盖。
- Modify: `packages/frontend/src/main.tsx`
  - 负责复用现有 `dataset.theme` 写入机制，并补充 `minimal` 所需的 localStorage 校验、`colorScheme` 映射和 Toaster light/dark 映射。
