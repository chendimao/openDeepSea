# 全局智能体库与对话式创建 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** 建立独立于项目和聊天室的全局智能体库，支持完整 CRUD、对话式创建草稿，并允许聊天室从全局库拉入智能体。

**Architecture:** 后端新增 `agents` 全局表和仓储，`room_agents` 增加 `global_agent_id` 作为聊天室成员关系。前端新增 `/agents` 管理页，聊天室邀请入口改为从全局智能体库拉入；执行 prompt 合并全局性格、规则与职责。

**Tech Stack:** Node.js、Express、better-sqlite3、TypeScript、React 18、Vite、TanStack Query、Tailwind CSS。
