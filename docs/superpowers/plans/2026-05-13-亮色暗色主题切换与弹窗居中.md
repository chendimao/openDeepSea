# 亮色暗色主题切换与弹窗居中 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** 增加 Harbor Light 默认亮色主题与暗色主题切换，并修复共享 Dialog 弹窗偏右下的问题。

**Architecture:** 使用前端本地状态和 `localStorage` 管理主题，根节点通过 `data-theme` 与 `dark` class 暴露当前主题。样式层以 CSS 变量为主，亮色变量放在 `:root`，暗色变量放在 `[data-theme="dark"]`；Dialog 使用独立 modal 动画避免覆盖居中 transform。

**Tech Stack:** React 18、TypeScript、Vite、Tailwind CSS、Radix Dialog、Sonner、Lucide React。

---

### Task 1: 主题状态与根组件接线

**Files:**
- Modify: `packages/frontend/src/main.tsx`
- Modify: `packages/frontend/src/components/AppShell.tsx`

- [x] **Step 1: 在 `main.tsx` 增加主题状态入口**

将根渲染逻辑改为 `RootApp` 组件，定义 `ThemeMode = 'light' | 'dark'`，初始化时读取 `localStorage.getItem('openclaw-room-theme')`。如果值不是 `dark`，默认 `light`。

- [x] **Step 2: 同步主题到根节点**

在 `RootApp` 内使用 `useEffect` 设置：

```ts
document.documentElement.dataset.theme = theme;
document.documentElement.classList.toggle('dark', theme === 'dark');
document.documentElement.style.colorScheme = theme;
localStorage.setItem('openclaw-room-theme', theme);
```

- [x] **Step 3: 让 `Toaster` 跟随主题**

将 `Toaster` 的 `theme` 属性从固定 `"dark"` 改为 `{theme}`，保持 `position="bottom-right"` 与 `richColors`。

- [x] **Step 4: 把主题状态传入 `AppShell`**

给 `AppShell` 增加 props：

```ts
type ThemeMode = 'light' | 'dark';

export function AppShell({
  children,
  theme,
  onThemeChange,
}: {
  children: ReactNode;
  theme: ThemeMode;
  onThemeChange: (theme: ThemeMode) => void;
}): JSX.Element
```

- [x] **Step 5: 在侧栏底部增加主题切换按钮**

将 `theme` 和 `onThemeChange` 传给 `ProjectRail`，在设置按钮上方加入一个按钮。亮色时显示 `Moon` 图标和“暗色”，点击切换到 `dark`；暗色时显示 `Sun` 图标和“亮色”，点击切换到 `light`。按钮沿用现有 `Settings` 按钮的样式。

### Task 3: 修复 Dialog 居中动画

**Files:**
- Modify: `packages/frontend/src/components/ui/Dialog.tsx`
- Modify: `packages/frontend/src/index.css`

- [x] **Step 1: 替换 `DialogContent` 的动画类**

将 `DialogContent` 的 `fade-up` 改为 `modal-pop`，保留 `fixed left-1/2 top-1/2 -translate-x-1/2 -translate-y-1/2`。

- [x] **Step 2: 添加 `modal-pop` keyframes**

在 `index.css` 增加：

```css
@keyframes modal-pop {
  from { opacity: 0; transform: translate(-50%, -48%) scale(0.98); }
  to { opacity: 1; transform: translate(-50%, -50%) scale(1); }
}

.modal-pop {
  animation: modal-pop 180ms cubic-bezier(0.16, 1, 0.3, 1) both;
}
```

同时在 `prefers-reduced-motion` 中包含 `.modal-pop`。

- [x] **Step 3: 更新 Dialog shadow 与 overlay**

`DialogPrimitive.Overlay` 使用 `bg-[var(--color-overlay)]`，`DialogContent` 使用 `shadow-[var(--shadow-dialog)]` 或 CSS class，避免暗色阴影固定污染亮色主题。
