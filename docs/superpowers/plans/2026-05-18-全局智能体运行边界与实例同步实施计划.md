# 全局智能体运行边界与实例同步 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 `/agents` 支持配置全局默认运行边界，允许群聊实例手动同步全局默认值，并让 `memory_scope` 真正控制记忆读取范围。

**Architecture:** 全局 agent 继续作为默认模板，room agent 继续作为运行副本。后端先补齐 `/agents` schema、room agent 同步接口和记忆读取边界；前端再补全类型、API、全局表单和群聊实例同步入口。

**Tech Stack:** Node.js、Express、Zod、SQLite、node:test、React 18、TypeScript、TanStack Query、Vite。

- [ ] **Step 3: 实现 repo 方法**

在 `packages/backend/src/repos/rooms.ts` 的 `roomAgentRepo` 中新增：

```ts
syncRuntimeDefaultsFromGlobal(id: string): RoomAgent | undefined {
  const existing = this.get(id);
  if (!existing) return undefined;
  if (!existing.global_agent_id) {
    throw new Error('agent has no global defaults');
  }
  const agent = agentRepo.get(existing.global_agent_id);
  if (!agent) {
    throw new Error('global agent not found');
  }
  const withAcp = this.setAcp(id, {
    acp_enabled: Boolean(agent.default_acp_backend),
    acp_backend: agent.default_acp_backend,
    acp_session_id: existing.acp_session_id,
    acp_session_label: existing.acp_session_label,
    acp_permission_mode: agent.default_acp_permission_mode,
    acp_writable_dirs: existing.acp_writable_dirs,
  });
  if (!withAcp) return undefined;
  return this.setCapabilitiesAndRuntime(id, {
    capabilities: withAcp.capabilities,
    default_runtime: agent.default_acp_backend ? 'acp' : 'none',
    runtime_backend: agent.default_runtime_backend,
    tool_policy: agent.default_tool_policy,
    workspace_policy: agent.default_workspace_policy,
    memory_scope: agent.default_memory_scope,
  });
}
```

注意：不要修改 `agent_name`、`agent_role`、`workflow_role`、`acp_session_id`、`acp_session_label`。

- [ ] **Step 4: 实现路由**

在 `packages/backend/src/routes.ts` 的 room agents 路由附近新增：

```ts
router.post('/rooms/:roomId/agents/:agentId/sync-global-runtime-defaults', (req, res) => {
  const existing = roomAgentRepo.get(req.params.agentId);
  if (!existing || existing.room_id !== req.params.roomId) return res.status(404).json({ error: 'not found' });
  try {
    const updated = roomAgentRepo.syncRuntimeDefaultsFromGlobal(req.params.agentId);
    if (!updated) return res.status(404).json({ error: 'not found' });
    wsHub.broadcast(updated.room_id, { type: 'room:agent_joined', roomId: updated.room_id, agent: updated });
    res.json(updated);
  } catch (err) {
    const message = (err as Error).message;
    if (message.includes('no global defaults') || message.includes('global agent not found')) {
      return res.status(409).json({ error: message });
    }
    res.status(400).json({ error: message });
  }
});
```

- [ ] **Step 3: 实现记忆读取计划函数**

在 `packages/backend/src/memory/context.ts` 新增：

```ts
import type { AgentMemoryScope, MemoryEntry, MemoryScope } from '../types.js';

export interface MemoryContextLoadPlan {
  includeDirectContext: boolean;
  includeCrossRoomRelevant: boolean;
  contextScopes: MemoryScope[];
}

export function resolveMemoryContextLoadPlan(scope: AgentMemoryScope | null | undefined): MemoryContextLoadPlan {
  switch (scope ?? 'room') {
    case 'project':
      return { includeDirectContext: true, includeCrossRoomRelevant: true, contextScopes: ['project', 'room', 'agent', 'task'] };
    case 'room':
      return { includeDirectContext: true, includeCrossRoomRelevant: false, contextScopes: ['room', 'agent', 'task'] };
    case 'agent':
      return { includeDirectContext: true, includeCrossRoomRelevant: false, contextScopes: ['agent', 'task'] };
    case 'task':
      return { includeDirectContext: true, includeCrossRoomRelevant: false, contextScopes: ['task'] };
    case 'none':
      return { includeDirectContext: false, includeCrossRoomRelevant: false, contextScopes: [] };
  }
}
```

再新增过滤函数：

```ts
export function filterMemoryEntriesByScope(entries: MemoryEntry[], scopes: MemoryScope[]): MemoryEntry[] {
  const allowed = new Set(scopes);
  return entries.filter((entry) => allowed.has(entry.scope));
}
```
