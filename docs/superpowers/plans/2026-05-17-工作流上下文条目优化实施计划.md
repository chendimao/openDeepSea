# 工作流上下文条目优化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增 `workflow_context_entries`，让 LangGraph 下游智能体只消费结构化工作流上下文，而不再默认拼接 `task_artifacts.content`。

**Architecture:** 保留原始 message、agent run、workflow step 和 artifact 作为审计数据；新增 workflow context repo 和 formatter 作为下游 prompt 的唯一工作流上下文来源。LangGraph 节点在阶段完成后写入上下文条目，后续节点通过 formatter 读取预算内摘要。

**Tech Stack:** Node.js、TypeScript、better-sqlite3、Express、Node test runner、LangGraph runtime。

---
