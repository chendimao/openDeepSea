# ACP 系统上下文检索一期 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现 ACP 可主动调用的只读系统上下文查询能力，并通过浏览器验证群聊消息触发 ACP 查询数据库事实后回复。

**Architecture:** 第一版只实现阶段 0/1：新增后端 `system-context` service 聚合现有 repo，新增 CLI/API 入口，修改 ACP 普通回复 prompt 明确要求系统状态问题先调用 CLI。精确事实查询走 repo，不使用摘要 prompt 或语义索引。

**Tech Stack:** TypeScript, Node.js, Express, SQLite/better-sqlite3, node:test, Vite/React browser smoke test.
