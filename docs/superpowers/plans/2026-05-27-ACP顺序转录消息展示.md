# ACP 顺序转录消息展示 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 移除阶段式混合默认展示，改为按 ACP `session/update` 顺序交错展示 assistant 正文 chunk 与工具调用，接近 Codex CLI transcript。

**Architecture:** 前端基于 `trace.events` 构建 chronological transcript。存在 `assistant_message` 事件时使用 transcript 渲染并避免重复最终正文；缺少 `assistant_message` 时回退为最终正文 + 完整 ACP timeline。

**Tech Stack:** React 18、TypeScript、Vite、Node test runner、现有 CSS。
