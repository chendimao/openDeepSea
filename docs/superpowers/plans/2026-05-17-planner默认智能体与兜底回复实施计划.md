# Planner 默认智能体与兜底回复 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 `planner` 固化为所有群聊默认且不可移除的内置智能体，并把聊天协作默认策略收敛为 `planner` 的兜底回复。

**Architecture:** 后端作为真实约束源，负责数据清理、默认设置、自动补齐 `planner` 和拒绝旧 routing mode。前端只展示合法选项，并把兜底智能体改成下拉单选。

**Tech Stack:** Node.js、Express、SQLite、TypeScript、React 18、Vite、TanStack Query。
