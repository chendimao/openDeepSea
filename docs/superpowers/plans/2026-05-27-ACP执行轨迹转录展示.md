# ACP 执行轨迹转录展示 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将现有 ACP timeline 渲染成更接近 Codex CLI 的紧凑执行轨迹，展示工具调用、读取文件、命令、diff、计划和调试事件。

**Architecture:** 复用后端已经落库的 `AgentTimelineEvent`，不改 ACP provider 协议。前端在 `AgentTimeline` 展示层新增 transcript 风格分组、摘要文案和更紧凑的事件正文，同时保持原有 raw/debug 折叠能力。

**Tech Stack:** React 18、TypeScript、Vite、现有 CSS、Node test runner。

---

**Files:**
