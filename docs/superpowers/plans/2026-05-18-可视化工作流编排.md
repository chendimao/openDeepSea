# 可视化工作流编排 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** 将现有固定 workflow 抽象成可版本化定义，支持可视化编排，并允许每个群聊选择不同 workflow。

**Architecture:** 新增 workflow definitions 仓储与 API，运行时从房间设置解析默认定义并保存定义快照。可视化页面使用 React Flow 编辑受控白名单节点和边，LangGraph 执行器从 definition 编译图，不开放任意代码执行。

**Tech Stack:** Node.js、Express、better-sqlite3、LangGraph、React 18、TypeScript、Vite、Tailwind、@xyflow/react。
