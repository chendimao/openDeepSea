# 工具调用详情按需加载 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 群聊消息列表默认不返回 `tool_call`、`tool_result`、`command_output` 的大字段，用户展开单条记录时再按需拉取完整详情。

**Architecture:** 后端在 `listForClientByRoom` 中返回轻量 trace event，保留时间线排序和摘要字段；新增 room-scoped trace event 详情接口按 `roomId/messageId/eventId` 返回完整事件。前端 `AgentTimelineItem` 在展开轻量事件时调用详情接口，并缓存已加载结果。

**Tech Stack:** Node.js + Express + SQLite backend, React 18 + TypeScript + Vite frontend, node:test 定向测试。
