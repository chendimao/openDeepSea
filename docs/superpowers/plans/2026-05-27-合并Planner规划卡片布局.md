# 合并 Planner 规划卡片布局实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**目标：** 将聊天消息中的「规划决策」摘要与下方 Planner 建议操作区合并为一张紧凑行动卡，减少重复信息和垂直占用，并移除「让我补充说明」按钮。

**架构：** 保持现有消息渲染链路不变，仅在前端展示层收敛信息结构。`RoomPage.tsx` 负责可执行 Planner 卡片，`MessageContent.tsx` 避免同一条消息重复渲染 JSON 规划摘要，`index.css` 调整结构化 JSON 摘要的兜底样式。

**技术栈：** React 18、TypeScript、Vite、Tailwind utility class、现有 `MessageContent` 结构化 JSON 渲染。
