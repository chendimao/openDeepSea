# 消息引用回复 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为群聊消息增加默认引用上一条可回复智能体消息、手动引用气泡消息、并将引用上下文传给智能体，避免多话题串话。

**Architecture:** 消息引用以 `metadata.reply_to` 持久化。前端 composer 负责展示默认/显式引用预览并随发送请求提交 `reply_to_message_id`；后端校验引用消息属于同一房间，写入引用摘要，并在 dispatch prompt 中注入“正在回复”的上下文。

**Tech Stack:** React 18、TypeScript、Express、SQLite、node:test。
