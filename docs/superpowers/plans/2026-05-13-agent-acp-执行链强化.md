# Agent/ACP 执行链强化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** 强化 OpenClaw Agent 与 ACP CLI 的真实执行链，支持 session spawn、执行状态、取消执行、stderr/error 展示和 CLI 输出解析。

**Architecture:** 后端新增执行记录与运行时控制层，用 SQLite 持久化执行状态，用内存 registry 管理正在运行的 CLI 子进程和 AbortController。OpenClaw Gateway 增加 session spawn 能力，ACP adapter 统一返回结构化 chunk/error/session 信息，前端通过 WebSocket 展示执行中、完成、失败和已取消状态。

**Tech Stack:** Node.js、Express、ws、better-sqlite3、child_process、AbortController、React 18、TypeScript、TanStack Query。

## Task 2: 运行时取消控制

**Files:**
- Create: `packages/backend/src/run-registry.ts`
- Modify: `packages/backend/src/routes.ts`

- [x] **Step 1: 创建运行时 registry**

创建 `packages/backend/src/run-registry.ts`：

```ts
const controllers = new Map<string, AbortController>();

export const runRegistry = {
  create(runId: string): AbortController {
    const controller = new AbortController();
    controllers.set(runId, controller);
    return controller;
  },
  get(runId: string): AbortController | undefined {
    return controllers.get(runId);
  },
  cancel(runId: string): boolean {
    const controller = controllers.get(runId);
    if (!controller) return false;
    controller.abort();
    return true;
  },
  remove(runId: string): void {
    controllers.delete(runId);
  },
};
```

- [x] **Step 2: 新增取消 API**

在 `packages/backend/src/routes.ts` 添加：

```ts
router.post('/agent-runs/:id/cancel', (req, res) => {
  const run = agentRunRepo.get(req.params.id);
  if (!run) return res.status(404).json({ error: 'not found' });
  const cancelled = runRegistry.cancel(req.params.id);
  if (!cancelled) return res.status(409).json({ error: 'run is not active' });
  const updated = agentRunRepo.updateStatus(req.params.id, 'cancelled');
  if (updated) wsHub.broadcast(updated.room_id, { type: 'agent_run:updated', roomId: updated.room_id, run: updated });
  res.json(updated);
});
```

- [x] **Step 3: 新增房间执行列表 API**

在 `routes.ts` 添加：

```ts
router.get('/rooms/:roomId/agent-runs', (req, res) => {
  res.json(agentRunRepo.listByRoom(req.params.roomId));
});
```
