# 子代理事件采集与原生编排 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让任务时间线能展示真实子代理 run，并在只有文本声明子代理时给出明确诊断。

**Architecture:** 保留 ACP `tool_call` 子代理归一化作为兼容层；新增 OpenDeepSea 原生 `agent_run_links` 关系，把平台编排的子代理 run 投影为任务 runtime event；前端在 ACP 时间线和任务时间线中展示父子 run 与诊断。

**Tech Stack:** Node.js、TypeScript、Express、SQLite、React 18、Vite、Node test runner。

- [ ] **Step 5: 实现 repo**

创建 `packages/backend/src/repos/agent-run-links.ts`：

```ts
import { nanoid } from 'nanoid';
import { db, now } from '../db.js';
import type { AgentRunLink, AgentRunRelationship, AgentRunLinkRole } from '../types.js';
```

实现：

```ts
export interface AgentRunLinkCreateInput {
  room_id: string;
  task_id?: string | null;
  parent_run_id: string;
  child_run_id: string;
  relationship: AgentRunRelationship;
  role: AgentRunLinkRole;
}

export const agentRunLinkRepo = {
  create(input: AgentRunLinkCreateInput): AgentRunLink {
    const id = nanoid();
    const createdAt = now();
    db.prepare(
      `INSERT INTO agent_run_links (
        id, room_id, task_id, parent_run_id, child_run_id, relationship, role, created_at
      ) VALUES (?, ?, ?, ?, ?, ?, ?, ?)`,
    ).run(
      id,
      input.room_id,
      input.task_id ?? null,
      input.parent_run_id,
      input.child_run_id,
      input.relationship,
      input.role,
      createdAt,
    );
    return this.get(id)!;
  },

  get(id: string): AgentRunLink | undefined {
    return db.prepare('SELECT * FROM agent_run_links WHERE id = ?').get(id) as AgentRunLink | undefined;
  },

  listByParentRun(parentRunId: string): AgentRunLink[] {
    return db.prepare(
      'SELECT * FROM agent_run_links WHERE parent_run_id = ? ORDER BY created_at ASC',
    ).all(parentRunId) as AgentRunLink[];
  },

  listByTask(taskId: string): AgentRunLink[] {
    return db.prepare(
      'SELECT * FROM agent_run_links WHERE task_id = ? ORDER BY created_at ASC',
    ).all(taskId) as AgentRunLink[];
  },
};
```

在 `packages/backend/src/routes.ts` 中为 task events 列表逻辑增加 helper：

```ts
function createSubagentRunTaskEvents(taskId: string): TaskEvent[] {
  const links = agentRunLinkRepo.listByTask(taskId);
  return links.flatMap((link) => {
    const childRun = agentRunRepo.get(link.child_run_id);
    const status = childRun?.status === 'failed'
      ? 'subagent_failed'
      : childRun?.status === 'completed'
        ? 'subagent_completed'
        : 'subagent_started';
    return [{
      id: `subagent-link:${link.id}:${status}`,
      task_id: link.task_id ?? taskId,
      room_id: link.room_id,
      seq: 0,
      type: 'runtime_event',
      layer: 'runtime',
      source_run_id: link.parent_run_id,
      payload: {
        timeline_type: status,
        timeline_status: childRun?.status === 'failed' ? 'failed' : childRun?.status === 'completed' ? 'completed' : 'started',
        parent_run_id: link.parent_run_id,
        child_run_id: link.child_run_id,
        child_agent_id: childRun?.agent_id,
        role: link.role,
        relationship: link.relationship,
      },
      created_at: link.created_at,
    }];
  });
}
```

该 helper 返回现有 `TaskEvent` 类型；如果编译器提示 `type` 或 `layer` 推断过宽，在返回对象后追加 `satisfies TaskEvent`，不要放宽类型定义。

```ts
async function runNativeSubagentExecution(input: {
  roomId: string;
  taskId: string;
  parentAgent: RoomAgent;
  runAgent: (input: TaskActionRunAgentInput) => Promise<TaskActionAgentResult>;
}): Promise<TaskActionStartResult> {
  const parentRunId = `task-action:${input.taskId}:subagent_execution:${Date.now()}`;
  const runIds: string[] = [];
  for (const role of SUBAGENT_EXECUTION_ROLES) {
    const agent = role.agentId === 'reviewer'
      ? selectDefaultPhaseAgent(input.roomId, taskRepo.get(input.taskId)!, 'verification')
      : selectDefaultPhaseAgent(input.roomId, taskRepo.get(input.taskId)!, 'subagent_execution');
    const result = await input.runAgent({
      agent,
      taskId: input.taskId,
      sourceMessageId: taskRepo.get(input.taskId)?.source_message_id,
      prompt: buildNativeSubagentPrompt(role.role, input.taskId),
    });
    if (result.runId) {
      runIds.push(result.runId);
      agentRunLinkRepo.create({
        room_id: input.roomId,
        task_id: input.taskId,
        parent_run_id: parentRunId,
        child_run_id: result.runId,
        relationship: 'subagent',
        role: role.role,
      });
    }
    if (result.status !== 'completed') {
      return {
        action: 'subagent_execution',
        status: 'failed',
        run_ids: runIds,
        blocked_reason: result.error,
      };
    }
  }
  return {
    action: 'subagent_execution',
    status: 'completed',
    run_ids: runIds,
  };
}
```
